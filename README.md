# CoffeeShop
A Webpage for writers
The website has the following functionalities:
1. sign in
2. sign up
3. post article
4. edit profile
5. view article
6. log out

dBConnect.php in the helperFunctions folder contains the php code for the database connection. 

Code was split into:
1. frontEnd: front End files
2. procFiles: processing files
3. js: javascript files
4. css: css files
5. helperFunctions
